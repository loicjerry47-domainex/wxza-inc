## Security

### Device Authentication
- **X.509 Certificates** (unique per device)
- **TLS 1.3** (all communications encrypted)
- **Secure Boot** (prevent firmware tampering)

### Data Privacy
- **No camera/microphone** (no privacy concerns)
- **Local processing** (animations run on-device)
- **Opt-in telemetry** (battery, usage stats)

### Compliance
- ✅ **CE** (European safety)
- ✅ **FCC** (electromagnetic interference)
- ✅ **RoHS** (hazardous substances)
- ✅ **WEEE** (e-waste recycling)
