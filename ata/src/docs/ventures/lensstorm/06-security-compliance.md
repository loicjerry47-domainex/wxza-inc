## Privacy Features

### Physical Camera Shutter

Mechanical shutter blocks lens when camera is off (100% guarantee no recording).

### LED Indicator

Always-on LED when camera/microphone active (cannot be disabled in software).

### Local Processing

95% of AI runs on-device (no cloud dependency for core features).

### Data Encryption

- **At Rest:** AES-256
- **In Transit:** TLS 1.3
- **User Data:** E2E encrypted (LENSSTORM cannot read)
